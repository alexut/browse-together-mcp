# TypeScript CLI Tool

A simple CLI tool for processing TypeScript files.

## Usage

```bash
deno run --allow-read cli.ts <file-or-directory-paths...>
```

## Development

Run using the task defined in `deno.json`:
```bash
deno task start <file-or-directory-paths...>
```
